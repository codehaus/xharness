Include

Description

Include dynamically includes/imports snippets of XML build files into another build file.

Parameters

Attribute Description Required
file The build file to include. Yes

Example

File build.xml:
      <!-- ... -->
      <target name="xharness.run">
          <property name="test.descriptor" value="acceptance.xml"/>
          
          <xharness name="art_svcs" resultsdir="../buld/results/xml">
              <include file="${test.descriptor}"/>
          </xharness>
          
          <!-- ... -->
      </target>
      <!-- ... -->

File acceptance.xml:
      <testgroup name="acceptance">
          <include file="configuration.xml"/>
          <include file="fps.xml"/>
          <include file="is2.xml"/>
          <!-- ... -->
      </testgroup>
