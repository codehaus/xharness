Skip

Description

Skip the TestCase or TestGroup the Skip Task is contained in, if a certain condition holds true, equivalent to the Ant Condition Task.

In addition to the Conditions defined in Ant, the Skip Task also supports XHarness File Conditions and Output Conditions.

Parameters

Attribute Description Required
description Description of the skip condition in text form. No

Parameters specified as nested elements

Skip supports all conditions supported by the Ant Condition Task. All conditions to test are specified as nested elements, for a complete list see here.

Examples

      <testcase name="foo_test">
          <skip description="Skipped on all platforms except Windows">
              <not>
                  <os family="windows"/>
              </not>
          </skip>
      </testcase>

skips the TestCase on all OS platforms except Windows.

      <testgroup name="bar_group">
          <skip description="Skipped in secure domains and on Irix">
              <or>
                  <contains string="${domain.name}" substring="secure"/>
                  <os name="irix"/>
              </or>
          </skip>
      </testgroup>

skips the TestGroup if the property domain.name contains the String "secure" or the OS name is Irix.
