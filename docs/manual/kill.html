Kill

Description

Kills a running background process. See also Xh-ExecBg and Xh-JavaBg Tasks.

Parameters

Attribute Description Required
processname Name of the process. Yes
failonerror Fail if the process can not be killed. Default is "true". No

Example:

  
       <xh-execbg processname="server" executable="myserver"/>
       
       <xh-exec executable="myclient"/>
       
       <kill processname="server"/>
